## Divine - Hello, World!
Divine is a framework for Python TUI Developments. Based on another
famous programming library called 'curses` which really lives up to
its name ;) Divine has **its own unique charms** and comes in **its
own unique asethetic** which somwhat similar to those terminals in 1990s.

### Potential Features_<
- **Object-Oriented** (Is this a feature or a bug?)
- **Resuable Templates**: such as Menus, Graphs, Grids 
- **Resuable Components**: such as Buttons, InputBoxes
- **Low-Level**: The more you can control, the better
